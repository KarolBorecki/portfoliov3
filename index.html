My Projects

Hamownia E

Hamownia E is the main measurement system used during testing of the TWR 2 rocket. The system was designed to provide a user-friendly interface for a wide range of measurement applications, and it also offers remote activation capabilities. This allows the mechanical team to safely perform temperature, pressure, and mass measurements, as well as control actuators and the pyrotechnic system from a safe distance away from the test stand.

FAVE

FAVE is a project developed as part of my engineering thesis. While choosing the topic, I wanted to learn something new, which led me to explore the field of fluid simulation. It is a program written in C/OpenGL that simulates a grid-based fluid environment. For visualization, I chose the marching cubes algorithm, as it was one of the most fascinating algorithms I encountered. It allowed me to generate a smooth, visually appealing approximation of the fluid surface in the simulation.

Twardowsky 2

Twardowsky 2 is the next iteration of a rocket project developed within the Students' Space Association. Building upon the experience gained from Twardowsky 1 — the first hybrid-propelled rocket successfully launched by the team — TWR 2 advances systems such as onboard electronics, aerodynamics, and ground support infrastructure, including the Hamownia E test stand. The project integrates various fields of engineering and serves as a platform for testing innovative solutions.

FOK

FOK is a steering rocket project developed within the Students' Space Association. My responsibilities included developing the onboard camera system and the onboard computer named Ganymede, built with an STM32 processor and running an RTOS system, which communicates with the advanced VectorNav inertial module. The system is responsible for collecting telemetry data and enabling real-time flight recording.

Crust Engine

Crust Engine is a fully functional 3D engine written in C++, utilizing various graphics libraries and the Eigen library. It was created to learn the fundamentals of computer graphics and perspective rendering. The engine supports .obj files, allows for the creation of simple games, and implements many classic 3D graphics algorithms. This project served as a hands-on introduction to the foundations of graphics engines and a deeper understanding of the rendering pipeline.

Guseppe

This project was developed for the Guseppe website as part of a marketing campaign. I co-created an interactive browser game that engaged users. Additionally, I was responsible for website maintenance, implementing new features in ReactJS, and ensuring smooth performance of the application.

Beer Me

Beer Me is a mobile application that allows users to scan a beer based on its photo and view detailed information about it. Users can save, comment on, and rate different beers. The project required building a custom database, image recognition model, and full application architecture. The backend was developed using Spring Boot, the frontend with Flutter, and a self-hosted Linux server was launched for testing, accessible via ZeroTier.

Fumbo

Fumbo is a 2D adventure game developed in Unity/C#, where the player takes on the role of a character rescuing trapped creatures. The gameplay focuses on exploration and puzzle-solving. I worked on this project in a small team, implementing many interactive features and mechanics, which helped me develop my teamwork and gameplay design skills.

Decode!

Decode! is a simple simulation game written in Python, where the player avoids black blocks and destroys them using bombs. The project was created as an experiment in mobile game development and publishing on the Google Play and App Store, giving me hands-on experience with the release process.

Augmented Reallity Campagin

A project developed for a client’s advertising campaign using Augmented Reality (AR). The application was showcased during trade fairs and allowed users to interact with products through AR, significantly increasing visitor engagement. It was a project that combined elements of marketing, UX, and mobile technologies.

Arena Clicker

Arena Clicker is a mobile clicker game developed in Python, where the player defeats monsters, levels up their character, and gains experience across increasingly challenging arenas. The game was publicly released on the Google Play / App Store and served as a great opportunity to learn about progressive game design and character development systems.

Pong

A classic Pong game recreated in Swift and published on the App Store / Google Play. The project had an educational purpose — it was my first step into the world of mobile applications and allowed me to get familiar with the iOS ecosystem and the app deployment process.
